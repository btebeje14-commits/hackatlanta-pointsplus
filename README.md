# PointsPlus – HackAtlanta

## Overview
PointsPlus is an AI-powered financial optimization platform built at HackAtlanta (sponsored by Cox and Nexlayer). The platform helps users maximize credit card rewards and airline loyalty points by analyzing spending habits, loyalty balances, and travel goals.

## Problem
Many users lose value from credit card rewards and airline points because they don’t know the best ways to earn or redeem them efficiently.

## Solution
PointsPlus provides personalized recommendations by analyzing:
- Credit card usage patterns
- Airline loyalty programs
- User travel goals

It then generates strategies to maximize rewards and optimize travel spending.

## Features
- Credit card reward optimization
- Airline points tracking
- Personalized travel planning
- AI-generated spending recommendations
